# Project Docs

Эта папка — центральное хранилище знаний о проекте ClutchStake.
Сюда добавляется документация, решения по архитектуре, описание фич и всё,
что помогает понимать контекст проекта.

## Структура

```
docs/
├── overview.md          # Общее описание проекта, цели, аудитория
├── architecture.md      # Архитектура системы, схемы, технические решения
├── features/            # Описание фич и флоу
│   └── ...
├── api/                 # Описание API контрактов
│   └── ...
└── decisions/           # ADR — важные архитектурные решения с обоснованием
    └── ...
```

## Как использовать

- Добавляй новые файлы по мере роста проекта
- AI-агент автоматически читает эту папку при работе над проектом
- Чем подробнее описание — тем точнее помощь
